Background Remover API Test

Test page untuk Background Remover API yang berjalan di http://127.0.0.1:5001

1. Download File Endpoint

Endpoint: POST /remove-background

Mengembalikan file PNG yang bisa langsung di download

2. Preview Endpoint

Endpoint: POST /remove-background-preview

Menampilkan hasil gambar langsung di browser (tanpa download)

3. Base64 Endpoint

Endpoint: POST /remove-background-base64

Menggunakan base64 untuk input dan output
